# SupertableDemo

SupertableDemo is an Angular-based application featuring a highly configurable **SuperTable** component. It leverages **PrimeNG** to provide a rich set of features for data presentation and manipulation.

## Setup Instructions

### Prerequisites
- [Node.js](https://nodejs.org/) (latest LTS version recommended)
- [Angular CLI](https://github.com/angular/angular-cli) version 18.2.21

### Installation
1. Clone the repository.
2. Navigate to the project directory:
   ```bash
   cd supertable-demo
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Development Server
Run the following command for a dev server:
```bash
npm start
```
Navigate to `http://localhost:4200/`. The application will automatically reload if you change any of the source files.

### Build
To build the project for production:
```bash
npm run build
```
